# مُفَرَّز (Mufarraz)

أداة ويب مستقلة لترتيب **فهرس الآيات** للمصححين اللغويين. تعمل بالكامل في المتصفح — لا خادم ولا رفع للملفات.

## الميزات

- قراءة ملف Word (.docx) بصيغة الإدخال (الآية + التخريج + رقم الصفحة)
- ترتيب حسب السورة ثم الآية ثم الصفحة
- دمج الصفحات المتعددة لنفس الآية داخل كل جزء
- تصدير ملف Word جاهز بصيغة الإخراج المجمّعة

## التشغيل محلياً

```bash
cd mufarraz
npm install
npm run dev
```

## البناء

```bash
npm run build
npm run preview
```

## الاختبارات

```bash
npm test
```

## النشر على GitHub Pages

1. أنشئ مستودعاً جديداً على GitHub (مثلاً `mufarraz`)
2. انسخ محتويات مجلد `mufarraz/` إلى جذر المستودع
3. من **Settings → Pages** اختر المصدر: **GitHub Actions**
4. ادفع إلى فرع `main` — سينشر الـ workflow تلقائياً

الرابط سيكون: `https://<username>.github.io/<repo-name>/`

> **ملاحظة:** GitHub Pages يستضيف مواقع ثابتة فقط. لذلك تُعالَج ملفات Word في المتصفح بـ JavaScript (JSZip) بدلاً من PHP/Laravel.
